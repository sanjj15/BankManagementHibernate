This project is a Java-based console banking application developed using Hibernate ORM to demonstrate object-relational mapping and seamless database integration. It enables users to perform core banking operations such as checking account balances, transferring funds between accounts, and managing account records. The application follows a layered architecture with DAO, Service, and Main modules to ensure proper separation of concerns. Hibernate efficiently manages database connectivity, transactions, and CRUD operations. Oracle Database is used as the backend for persistent data storage. Transaction management is implemented to maintain data consistency and integrity during fund transfers. Overall, the project showcases a practical, real-world implementation of a database-driven application using Java and Hibernate.
